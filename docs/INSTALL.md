# Setup guide

1. Clone this repository
```bash
$ git clone https://github.com/eXoterr/eShop
```

2. Open cloned repo
```bash
$ cd eShop/
```

3. Create default config
```bash
$ cp ./.env.example ./.env
```

4. Edit configuration

5. Install dependencies
```bash
$ npm install
```

6. Run application
```bash
$ npm run dev
```
